# HACKTOBERFEST-2022-RESTAPI-

Create a Fully working REST API in Node JS

## How to install the dependencies

```bash
clone the repo
$ cd HACKTOBERFEST-2022-RESTAPI
$ cd RESTAPI
$ npm i
$ npm run dev
```
